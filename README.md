# 🧠 NeuroLearn AI — The Future of Adaptive Learning

[![Live Demo](https://img.shields.io/badge/Live-Demo-818CF8?style=for-the-badge&logo=vercel)](https://neurolearnai-portal.vercel.app)
[![Backend Status](https://img.shields.io/badge/Backend-Live-34D399?style=for-the-badge&logo=render)](https://neurolearn-ai-b0vm.onrender.com)

**NeuroLearn AI** is a production-hardened, full-stack adaptive learning platform designed to help students master complex subjects through artificial intelligence. By combining advanced retention algorithms with Large Language Models (LLMs), it converts passive reading into active mastery.

---

## 🚀 Core Intelligence Features

### 🪄 AI Knowledge Extractor (Vision-Powered)
- **Multi-Image Support**: Upload up to 5 photos of textbooks or handwritten notes.
- **Auto-Generation**: Instantly converts images/PDFs into structured Flashcards and Quizzes.
- **Smart Parsing**: Leverages Llama 3.2 Vision for high-accuracy OCR and content extraction.

### 🤖 Personal AI Study Coach
- **Retention Tracking**: Calculates your "Memory Decay" and tells you exactly when to revise.
- **Dynamic Guidance**: Provides personalized study tips based on your recent quiz performance.
- **Interactive Tutor**: A dedicated AI Tutor to explain complex topics 24/7.

### 🛠️ Professional Admin Portal
- **Real-time Metrics**: Monitor platform growth, student registrations, and global study volume.
- **Activity Stream**: Live feed of student achievements and quiz scores.
- **User Management**: Comprehensive directory for managing learner accounts and roles.

---

## 💻 Tech Stack

- **Frontend**: React 18, Vite, Vanilla CSS (Premium Glassmorphism Design).
- **Backend**: Node.js, Express, MongoDB Atlas.
- **AI Engine**: Groq API (Llama 3.3 Versatile), Llama 3.2 Vision.
- **Intelligence**: Custom Spaced Repetition Algorithm for retention scoring.
- **Security**: JWT-based Authentication, Role-Based Access Control (RBAC).

## 🛠️ Local Setup

1. **Clone the repo**:
   ```bash
   git clone https://github.com/AyushVishwakarma-CodeHub/NeuroLearn-AI.git
   ```
2. **Install Dependencies**:
   ```bash
   npm run install-all
   ```
3. **Set Environment Variables**:
   Create a `.env` file in the `server` folder with your `MONGO_URI` and `GROQ_API_KEY`.
4. **Launch Everything**:
   ```bash
   npm run dev
   ```
